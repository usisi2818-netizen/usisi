08 / LOG · 기록 타임라인

규격 — 한 줄에 하나씩 날짜|제목|설명. 위에서부터 순서대로 표시. 비우면 기본값.
⭐ 여기에 D-Day를 넣고 싶으면 설명 부분에 {생일} 또는 {데뷔} 라고 중괄호째로 써 주세요 → 화면에서 D-33 · D+102 처럼 자동으로 바뀝니다.
예) 2026. 06. 01|BIRTHDAY|다음 생일까지 {생일}
(숫자를 직접 쓰면 자동 계산이 안 되고 그냥 글자로 남아요)

11 / 사용법 · 규격 안내

· 프사 — SOOP 아이디만 넣으면 자동 연동, 직접 URL을 넣으면 그게 우선.
· D-Day / D+ — 기본정보 카드의 생일·데뷔일 칸에서만 계산돼요. 연도는 있어도 없어도 인식(06.01·2026.06.01 둘 다 OK).
· LOG 기록에 D-Day 넣기 — 설명에 {생일}·{데뷔}를 중괄호째 쓰면 자동 숫자로 바뀌어요.
· 능력치/일정/기록 — 각 카드의 규격(이름:숫자, 아이콘|시간|라벨, 날짜|제목|설명) 그대로. 비우면 기본값 유지.
· 옷장 사진 — 세로 3:4 (900×1200 권장), SOOP 비공개 게시판 업로드 → 이미지 우클릭 → "이미지 주소 복사" (https://stimg.sooplive.com/…) — 글 주소 ✗.
· VOD — 다시보기 주소의 숫자 ID만.
· 저장 후 안 보이면 — 해당 페이지에서 Ctrl+Shift+R 강력 새로고침.
